# PPS-DeadCells

This project is aimed to develop a rogue-lite games based on the famous DeadCells

## Build
The system uses Sbt as build system and dependency manager.

To build:
```
$ git clone https://github.com/LucaBazzo/pps-deadcells.git
$ cd pps-deadcells
$ sbt compile
```

To test:
```
$ sbt test
```

To execute:
```
$ sbt run
```
